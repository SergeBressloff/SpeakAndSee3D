Quick Start & Shortcuts

Tip: You can press F1 anywhere in the app to open this Help. When typing in a text box, the global shortcuts are paused until you exit typing mode.

Quick Start

Use ← and → to switch between Load and Generate.

Load mode lets you browse existing 3D models.

  • Press T to enter text input.
  • Type your prompt and press Enter to search.
  • Press D to delete the current 3D model.

Generate mode lets you create new 3D content via prompts.

  • Use ↑ and ↓ to cycle diffusion models.
  • Press Space any time (when not typing) to start/stop voice recording.

  • Press S to save the generated 3D model.

You can also click outside the text box to leave typing mode.

Tips

  • If a shortcut doesn’t work, make sure the focus isn’t inside a text field.
  • Voice input requires microphone access on your system.
  • 3D models are saved next to the main executable in: data\viewer_assets\3d_assets.
  • If running on CPU, use LCM_Dreamshaper_v7 image diffusion.
  • If running on GPU, use flux_1_schnell image diffusion.

Troubleshooting

  • No audio captured? Check your OS microphone permissions and input device selection.
  • Can’t switch modes? Ensure you’re not in typing mode (press Esc).
